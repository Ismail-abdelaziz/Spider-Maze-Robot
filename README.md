# Spider Maze Robot 🕷️

## Project Idea
This project aims to design and build a hardware-based robotic spider that can navigate through a maze from the start point to the end point.

## Description
The robot uses sensors to detect walls and obstacles, allowing it to make decisions and move in the correct direction. A microcontroller is used to control the movement and process the sensor data.

## Objectives
- Build a spider-like robot using hardware components
- Navigate through a maze autonomously
- Apply basic robotics and embedded systems concepts

## Components
- Microcontroller (Arduino Nano)
- Ultrasonic 
- Servo Motors
- Power Supply
- Chassis (Spider design)

## Features
- Autonomous movement
- Obstacle detection
- Decision making inside maze
